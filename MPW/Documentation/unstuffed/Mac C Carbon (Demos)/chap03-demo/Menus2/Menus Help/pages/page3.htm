CREATING A TITLE PAGE

    A complete set of Help content is referred to as a book. Apple Help lets you provide access to your Apple Help book via the relevant item in the Help menu (and also via "?" buttons in your application).

    You must create a title page for your help book. The title page is an HTML file, with a creator of 'hbwr', placed at the root level of your Help book folder. (Your Help book folder, or an alias to it, must, in turn, be placed in the Help folder in the System folder (Mac OS 8/9) and in ~/Library/Preferences/Apple Help Prefs/ (Mac OS X).)

    On Mac OS 8/9, the title page specifies how your book is listed in the Help Center, which is opened when you choose Help Center from the Mac OS 8/9 Help menu.

1 - Specifying a Title

    You must specify a title for your book using the AppleTitle meta tag in the HTML header of your title page file. This title is displayed in the Mac OS 8/9 Help Center listing. The following example specifies the title "Menus Help": 

      <meta name="AppleTitle" content="Menus Help">
2 - Specifying an Icon

    You can use the AppleIcon meta tag in your title page file's HTML header to specify an icon for your book's Help Center listing. The following example specifies an icon called bookicon.gif located in the "graphics" folder in the Help book folder titled "Menus Help": 

      <meta name="AppleIcon" 
    content="Menus%20Help/graphics/bookicon.gif">

    The icon should be 16 by 16 pixels in size with a transparent background, and saved as a Graphics Interchange Format (GIF) file. The path to the icon should be specified relative to the Help folder.

    If you do not specify an icon, Apple Help provides a default icon.

3 - Checking Mac OS 8/9 Help Center Presentation

    With the title page file created and located in your Help book folder, and with your Help book folder (or an alias to it) located in the Help folder in the System folder, you may check that it presents correctly in the Mac OS 8/9 Help Center by choosing Help Center from the Finder's Help menu.
