CREATING AN ITEM IN THE MAC OS 8/9 HELP MENU

    The following applies to Mac OS 8/9 only.

    To install an item in the Mac OS 8/9 Help menu, you must create a simple Apple Guide file using the Guide Maker 2.0 tool provided in the Apple Help SDK.

1 - Creating a Guide Script Source File

    The first step is to create a Guide Script source file containing the App Creator command and specifying a menu item or items with the Help Menu command. Here is the Guide Script source file used for the "Menus Guide" file: 

      <Help Menu>"Menus Help",help,"?"
  <App Creator>'????'
  <Startup Window> presentation, "Empty Sequence"
  <Balloon Menu Text>"Displays basic information on
    providing user assistance with Apple Help."
  <Define Sequence>"Empty Sequence"
  <Define Panel>"Empty Panel"
  Empty Text
  <End Panel>
  <End Sequence>
2 - Creating a Resource File

    The second step is to create a resource file containing two resources:

    • 'html' ID 1000. This resource will tell Apple Guide that the Apple Guide file contains a menu item that should be redirected to HTML help content.
    • 'TEXT'. The name of this resource must match the name displayed in the Help Menu command in the Guide Script source file. The content of this resource is passed to the Help Viewer when the user selects this menu item from the Help menu. (In the resource file for the "Menus Guide" file, this resource is named "Menus Help" and contains the text "Menus%20Help/Menus.htm". As will be seen, "Menus.htm" is an HTML file - specifically, the title page file - which is located in a folder titled "Menus Help".)
    The name of the resource file must end in ".rsrc". (For the "Menus Guide" file, the resource file was named "Menus Guide.rsrc".)
3 - Compiling the Apple Guide File

    The third step is to compile the Apple Guide file. Launch the application Guide Maker 2.0 and proceed as follows:

    • Import the Guide Script Source File. Choose Import Source from the File menu and import the Guide Script source file. A new Guide Maker file will be created.
    • Save the New Guide Maker File. Choose Save from the File menu, and save the Guide Maker file. (For the "Menus Guide" file, the Guide Maker file was named "Menus Guide.gm".)
    • Compile the Apple Guide File. Choose Compile from the File menu and name the compiled Apple Guide file. (The Menus Apple Guide file was named "Menus Guide".) When the file is compiled, Guide Maker 2.0 looks for a resource file with the same name as the Apple Guide file, but ending in ".rsrc". If such a file is found, its resources are copied to the resulting Apple Guide file.
4 - Checking the Application's Help Menu

    Drag the newly-created Apple Guide file to the folder containing your application. Launch your application and click on the Help menu. You should see an appended item in that menu with the name specified at the Help Menu command in the Guide Script source file (and repeated in the name of the 'TEXT' resource in the resource file).
