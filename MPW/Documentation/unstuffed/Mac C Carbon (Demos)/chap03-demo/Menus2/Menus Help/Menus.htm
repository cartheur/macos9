Despite the above title, the following does not provide user assistance for the Menus2 demonstration program as such. Rather, it provides a brief description of how to provide user assistance for your application using Apple Help.
Apple Help is an update to, and enhancement of, previous versions of the Mac OS help system. It supports all features of Apple Guide, and is available on Mac OS 8.6 and later.
Apple Help allows you to display HTML-based help content. This content is displayed by the Help Viewer, a lightweight browser which is launched automatically when required. The Help Viewer is optimized to take advantage of AppleScript and the Sherlock search engine through the use of special HTML tags.
Apple Help documentation and tools are included in the Apple Help Software Development Kit (SDK), which is available at href="http://developer.apple.com/sdk .
