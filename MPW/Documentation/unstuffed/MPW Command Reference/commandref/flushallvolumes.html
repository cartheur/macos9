FlushAllVolumes

Built-in

SYNTAX

FlushAllVolumes

DESCRIPTION

The FlushAllVolumes command allows the user to explicitly flush all volumes, overriding the settings of the DontFlushVolumes and DontFlushServers preferences.

This built-in command was added in the v. 3.5 MPW Shell.

INPUT

None

OUTPUT

None

STATUS

FlushAllVolumes always returns a status code of 0.

PARAMETERS

None

OPTIONS

None
