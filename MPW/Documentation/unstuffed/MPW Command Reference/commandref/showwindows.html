ShowWindows

Built-in

SYNTAX

ShowWindows window…

DESCRIPTION

The ShowWindows command causes the specified windows, if previously hidden, to become visible.

This built-in command was added in the v. 3.4.2 MPW Shell.

INPUT

None

OUTPUT

None

STATUS

ShowWindows can return the following status codes:

0

no errors

1

parameter or option error

PARAMETERS

window…

Specifies one or more windows to be shown.

OPTIONS

None
