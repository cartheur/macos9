AboutBox

Tool

SYNTAX

AboutBox

DESCRIPTION

The AboutBox command has the same effect as choosing About MPW from the Apple menu when MPW is the active application. The standard MPW About dialog box, showing the version number, is displayed.

INPUT

None

OUTPUT

None

STATUS

AboutBox always returns a status code of 0.

PARAMETERS

None

OPTIONS

None
