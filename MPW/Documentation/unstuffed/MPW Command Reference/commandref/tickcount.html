TickCount

Built-in

SYNTAX

TickCount

DESCRIPTION

The TickCount command returns the current system tick count. This command has no parameters. It simply performs a Tickcount() call and returns the result to standard output. For more information, refer to Inside Macintosh: Macintosh Toolbox Essentials.

INPUT

None

OUTPUT

Standard output.

STATUS

None

PARAMETERS

None

OPTIONS

None

EXAMPLES
